E-Commerce Backend

A scalable and secure backend application for an E-Commerce platform built using Spring Boot.
This project provides REST APIs for product management, authentication, cart handling, order processing, and user management.

Features
Authentication & Authorization
User Registration
User Login
JWT Authentication
Role-Based Authorization (Admin/User)
Password Encryption using BCrypt
Secure REST APIs
Product Management
Add Product
Update Product
Delete Product
View Products
Search Products
Category Management
Cart Management
Add to Cart
Remove from Cart
Update Quantity
View Cart Items
Order Management
Place Orders
Order History
Order Status Tracking
User Features
User Profile Management
Address Management
Additional Features
Global Exception Handling
Validation Handling
RESTful API Design
MySQL Database Integration
Tech Stack
Java
Spring Boot
Spring Security
JWT (JSON Web Token)
Hibernate / JPA
MySQL
Maven
Lombok
